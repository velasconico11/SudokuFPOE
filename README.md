# Sudoku 6x6 - FPOE 2026-1

Juego de Sudoku desarrollado como miniproyecto para la materia
Fundamentos de Programación Orientada a Eventos.

## ¿De qué trata?
Un Sudoku de 6x6 donde hay que completar la cuadrícula con números
del 1 al 6 sin repetir en filas, columnas ni bloques de 2x3.

## ¿Cómo se juega?
1. Dale al botón JUGAR para iniciar
2. Haz clic en una celda y escribe un número del 1 al 6
3. Si el número es incorrecto la celda se pone roja
4. Usa el botón AYUDA si te atascas (puede resolver hasta la penultima casilla)
5. ¡Completa el tablero para ganar!
